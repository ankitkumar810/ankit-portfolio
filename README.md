# Ankit Kumar - Personal Portfolio 🚀

A premium, production-ready personal portfolio website built for a Software Development Engineer role. Featuring a modern dark theme, glassmorphism, smooth animations, and a fully responsive design.

## 🔗 Live Demo
[View Live Website](https://ankitkumar810.github.io/ankit-portfolio)

## ✨ Features
- **Modern UI/UX**: Dark theme with premium glassmorphism effects.
- **Interactive Background**: Dynamic particles.js integration.
- **Custom Cursor**: Smooth dot + outline cursor with interactive states.
- **Animated Skills**: Synchronized skill bar animations triggered on viewport entry.
- **Project Showcase**: Filterable project gallery with hover effects.
- **Certificate Gallery**: Professional cards with direct links to Google Drive assets.
- **Responsive Design**: Mobile-first approach, optimized for all devices.
- **SEO Optimized**: Meta tags, Open Graph, Twitter Cards, and JSON-LD structured data.
- **Performance**: Lazy-loaded animations and optimized asset structure.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure and SEO metadata.
- **CSS3**: Custom variables, Flexbox/Grid, and complex animations.
- **JavaScript (ES6+)**: Intersection Observer API, DOM manipulation.
- **Libraries**: [Particles.js](https://vincentgarreau.com/particles.js/), [Font Awesome](https://fontawesome.com/).

## 🚀 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/ankitkumar810/ankit-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ankit-portfolio
   ```
3. Open `index.html` in your browser or use a local server:
   ```bash
   # If using VS Code, right-click index.html and select 'Open with Live Server'
   # Or using Python:
   python3 -m http.server 8080
   ```
